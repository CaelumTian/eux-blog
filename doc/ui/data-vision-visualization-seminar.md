数据可视化，是一门随着技术发展的科学艺术 在数据纪录，存储，计算都不发的的时期，数据可视化仅仅停留在信息可视化时期。随着社会进入互联网时代，云计算，云储存，大数据的出现，如何读懂数据，利用数据，成了这个时代的关键，可视化设计也成了这个读懂数据的利器。 这次分享会，主要自己的一些经验和经典案例。
